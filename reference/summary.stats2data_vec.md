# Summarize a stats2data Descriptives result

Computes target-vs-simulated statistics and RMSE for a `stats2data_vec`
object.

## Usage

``` r
# S3 method for class 'stats2data_vec'
summary(object, ...)
```

## Arguments

- object:

  An object of class `stats2data_vec`.

- ...:

  Additional arguments (unused).

## Value

An object of class `summary.stats2data_vec`, printed by
[`print.summary.stats2data_vec`](https://sebastian-lortz.github.io/stats2data/reference/print.summary.stats2data_vec.md).
