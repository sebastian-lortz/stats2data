# Summarize a stats2data MLR result

Computes target-vs-simulated statistics and RMSE for a `stats2data_mlr`
object.

## Usage

``` r
# S3 method for class 'stats2data_mlr'
summary(object, ...)
```

## Arguments

- object:

  An object of class `stats2data_mlr`.

- ...:

  Additional arguments (unused).

## Value

An object of class `summary.stats2data_mlr`, printed by
[`print.summary.stats2data_mlr`](https://sebastian-lortz.github.io/stats2data/reference/print.summary.stats2data_mlr.md).
